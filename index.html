Feladat

Írj egy javascript kódot, mely a HTML5 canvas-re kirajzol egy piros négyzetet!

Help

Az alábbi kulcsszavakat érdemes használni: document, getElementById(), getContext('2d'), moveTo(x,y), lineTo(x,y), stroke(), rect(x,y,w,h), fillStyle, fill()

More help

Itt egy zöld kört rajzoló kód: var canvas = document.getElementById('myCanvas'); var context = canvas.getContext('2d'); context.beginPath(); context.arc(95,50,40,0,2*Math.PI); context.fillStyle= "#3ef25f"; context.fill();
